# Set-O365MailboxPermission

## SYNOPSIS
Add or remove mailbox permissions on an Office 365 account; Full, Send As, or Send on Behalf

## SYNTAX

```
Set-O365MailboxPermission [-delegate_account] <Object> [-permission] <Object> [-set_type] <Object>
 [-target_account] <Object> [<CommonParameters>]
```
