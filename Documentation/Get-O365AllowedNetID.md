# Get-O365AllowedNetID

## SYNOPSIS
Gets list of NetIDs that have access to administer the Service Account; these NetIDs also have full mailbox access

## SYNTAX

```
Get-O365AllowedNetID [-addr] <Object> [<CommonParameters>]
```
